## 💡 About

### Techs:
- **Reactjs**, **Typescript**;

## 📁 Project

- Construct a Landing page interactive;
- Page made for task control;

## 💡 Functionalities

- Add a new task;
- Remove a task;
- Mark a task as done;
- Control of tasks done and all tasks;

## 👨🏽‍💻 To use
### Steps:
- git clone https://github.com/jmoura-dev/todo-list.git
- cd todo-list
- npm i
- npm run dev
